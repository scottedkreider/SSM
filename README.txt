1. Clone this github repo with github desktop
2. (optional) Open code with VSCode from github desktop
3. Install Node.js / NPM on your machine
    https://nodejs.org/en/download/
4. Use command line to navigate to the folder (Users -> Documents -> Github -> SSM)
5. (first time / maybe) run "npm install" in the command prompt
6. Run "npm start" in the command prompt
7. Navigate to localhost://5000
